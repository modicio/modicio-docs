    Fork this Definition according to the overall fork specification.

    Fork this Definition according to the overall fork specification. This operation produces a deep copy of the Definition and all its Rules.

    This operation takes the identity parameter because ParentRelationRules target a specified identity which must be exchanged during the fork.

    This operation calls Rule.fork(identity) on each rule.

    identity

    the identifier of the forked ModelElement following the default usage.

    returns

    Definition - the forked deep-copy of this Definition
