ModelElementData

case class ModelElementData(name: String, identity: String, isTemplate: Boolean, variantTime: Long, runningTime: Long, versionTime: Long, variantId: String, runningId: String, versionId: String) extends Product with Serializable

Tuple to represent a ModelElement for serialisation.

This class is produced by the ModelElement itself.

Name and identity form a unique combination which serves as ID.

name

the name of the type-modelElement

identity

identity value

isTemplate

flag if the ModelElement describes a template-type or instantiable-type
