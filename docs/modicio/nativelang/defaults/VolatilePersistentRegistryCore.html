VolatilePersistentRegistryCore

class VolatilePersistentRegistryCore extends AnyRef

Helper class for VolatilePersistentRegistry. Includes all required buffers and locks. Can be used to have several VolatilePersistentRegistries use the same buffers and locks.

This class is used for testing and benchmarking mainly.
